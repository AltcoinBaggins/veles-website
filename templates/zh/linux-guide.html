Linux dVPN Guide

安装

如果您使用的是Linux，根据您的发行版，可以使用多种工具。 当然，最普遍的连接方式是使用OpenVPN软件。

在Ubuntu或Debian上，您可以使用这样的命令

  • sudo apt-get update
  • sudo apt-get install openvpn

在CentOS上，您可以启用EPEL存储库，然后通过以下命令安装：

  • sudo yum install epel-release
  • sudo yum install openvpn

配置

下载Veles配置文件并输入以下内容移动到其存储库：

  • wget https://explorer.veles.network/dapi/getOpenVPNConfig
  • sudo mv /etc/openvpn/client/veles.ovpn


防止DNS泄漏，请使用此命令（本例中使用的IP仅作为示例，需要替换为新主节点列表中的IP）

  • echo -e "#Veles decentralized DNS\nnameserver 111.111.111.111"

对于GUI用户，请遵循以下步骤。
首先用鼠标右键单击网络管理器图标，然后单击编辑连接

现在，我们将点击“+”符号添加新连接

我们将选择“导入保存的配置文件...”导入我们的配置

现在找到我们的配置位置，在我们的例子中，它是在/etc/openvpn/client/并选择veles.ovpn

您不需要干其他事情了，它会自动为您创建Veles VPN配置文件，您只需按“保存”


连接

对于CLI用户，请遵循以下步骤。
现在，只需将openvpn指向配置文件，就可以连接到VPN：

  • sudo openvpn --config /etc/openvpn/client/veles.ovpn

对于GUI用户，请遵循以下步骤。
用鼠标右键单击网络管理器的图标打开它。在这里您将看到“VPN连接”。点击它。标记“veles”并享受您的连接！

Installing Stunnel

This explains the process of installing Stunnel as a client in Linux.

On Ubuntu or Debian, you can install it just as you did on the server by typing:

  • sudo apt-get install stunnel

On CentOS you can install Stunnel by typing:

  • yum install stunnel

Configuring Stunnel

Download Veles Stunnel configuration files and move to it's repositories by typing:

  • wget https://explorer.veles.network/dapi/getStunnelConfig
  • wget https://explorer.veles.network/dapi/getStunnelCertificate
  • wget https://explorer.veles.network/dapi/getShieldedOpenVPNConfig
  • sudo mv shielded-veles.ovpn /etc/openvpn/client/
  • sudo mv veles.stunnel.conf /etc/stunnel/stunnel.conf
  • sudo mv stunnel.pem /etc/stunnel/

Connecting OpenVPN Client with Stunnel

For CLI Users follow this steps.
Now, you can connect to the VPN trough Stunnel by just turning on Stunnel and pointing the OpenVPN command to the client configuration file:

  • sudo stunnel
  • sudo openvpn --config /etc/openvpn/client/veles-shield.ovpn

For GUI Users follow this steps.
Follow steps from OpenVPN guide and import veles-shield.ovpn and don't forget turn on Stunnel

Installing Obfuscation Proxy

This explains the process of installing ObfsProxy as a client in Linux.

On Ubuntu or Debian, you can install it just as you did on the server by typing:

  • sudo apt-get install obfsproxy

Configuring Stunnel

Download Veles configuration files and move to it's repositories by typing:

  • wget https://explorer.veles.network/dapi/getShieldedOpenVPNConfig
  • sudo mv shielded-veles.ovpn /etc/openvpn/client/

Connecting OpenVPN Client with ObfsProxy

For CLI Users follow this steps.
Now, you can connect to the VPN by just turning on ObfsProxy and pointing the OpenVPN command to the client configuration file:

  • sudo obfsproxy scramblesuit --dest 80.211.132.243:21343 --password=SSSTIZ3LG5SSS43OMSSSM4LKGQFASSSS client 127.0.0.1:21337
  • sudo openvpn --config /etc/openvpn/client/veles-shield.ovpn

We will use this password only in pre-alpha testing phase. In futer there will be used random value.


For GUI Users follow this steps.
Follow steps from OpenVPN guide and import veles-shield.ovpn and don't forget turn on ObfsProxy

测试您的VPN连接

一旦所有的东西都安装好了，只需要一个简单的检查就可以确认他是否运作。在未启用VPN连接的情况下，打开浏览器并转到 DNSLeakTest。
该站点将返回由您的网络服务提供商分配的IP地址，以及您在世界其他地方显示的IP地址。若要通过同一网站检查您的DNS设置，请单击扩展测试（Extended Test），它会告诉您正在使用的DNS服务器。
现在，连接到您的VPN客户端并刷新浏览器，应该会显示与您的VPN服务器的完全不同的IP地址。然后再次进行扩展测试（Extended Test）检查您的DNS设置，并确认您正在使用Veles dVPN推送的DNS解析。
