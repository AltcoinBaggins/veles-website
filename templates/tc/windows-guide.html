Windows dVPN 教程

安裝

可在OpenVPN的下載頁上找到適用於您Windows的OpenVPN客戶端。

OpenVPN需要管理員許可權才能安裝。



配置 OpenVPN

安裝OpenVPN後，下載配置檔
，並且將檔複製到你的配置檔夾中，例如：

  • C:\Program Files\OpenVPN\config

當您啟動OpenVPN時，它將自動尋找配置檔並啟動他。

OpenVPN每次使用時都必須以管理員身份運行，您不必每次使用OpenVPN時右鍵單擊並選擇以管理員身份運行，您可以對此進行預設，但這必須通過管理帳戶進行。這也意味著標準用戶需要輸入管理員密碼才能使用OpenVPN。另一方面，除非客戶端上的OpenVPN具有管理員許可權，否則標準用戶無法正確連接到伺服器，因此需要提升許可權。

如果要將OpenVPN設置為始終以管理員身份運行，請右鍵單擊其快捷方式圖示，然後選擇屬性。在相容性選項卡的底部，單擊按鈕為所有用戶更改設置。在新窗口中，選以管理員身份運行。



連接

每次啟動OpenVPN GUI時，Windows都會詢問是否允許程式對電腦進行更改單擊是（yes）。啟動OpenVPN客戶端應用程式後，他會隱藏在右下角託盤中，以便可以根據需要連接和斷開VPN，事實上他不會默認進行VPN連接。

進入右下角系統託盤並右鍵單擊OpenVPN圖示，啟動連接。這將打開菜單。在菜單頂部選擇veles（這是我們的veles.ovpn配置檔），然後選擇連接。

在建立連接時，將打開一個狀態窗口，顯示日誌輸出，並在連接客戶端後顯示一條消息。

以同樣的方式斷開與VPN的連接：進入右下角系統託盤，右鍵單擊OpenVPN小圖示，選擇客戶端配置檔，然後單擊斷開連接。

安裝 Stunnel

按照以下步驟下載並安裝Stunnel客戶端：

Stunnel需要管理權限才能安裝。

下載官方的Stunnel 客戶端 右鍵點擊小圖示 以管理員身份運行.

選擇 "同意".

選擇 “為任何人使用此電腦的安裝” 。

在這塊，什麼都不要更改。

選擇安裝的目錄。

在安裝結束時，它會打開新的控制臺窗口。 您可以輸入任何資訊，也可以留空。 怎麼做都沒關係，對您的連接沒有任何影響。

不要勾選“安裝完成開始使用隧道”，只需單擊“完成” 就可以。



配置 Stunnel

下載 Stunnel 客戶端配置, Stunnel 客戶端證書 .

下載完成後，進入Stunnel配置目錄，您可以刪除所有檔，並且複製之前下載的檔到這個目錄裏面 veles.stunnel.conf和 stunnel.pem

可能需要將veles.stunnel.conf重命名為 stunnel.conf


  • C:\Program Files\Stunnel\config\

現在下載 隱蔽的OpenVPN配置 檔並且複製檔到:

  • C:\Program Files\OpenVPN\config\


將OpenVPN客戶端與Stunnel連接

首先打開Stunnel。 然後打開OpenVPN客戶端，右鍵單擊OpenVPN小圖示來啟動連接。 打開菜單，選擇菜單頂部的 veles-shielded （這是我們之前下載的 veles-shielded.ovpn 配置檔），然後 連接 。

建立連接後，將打開狀態窗口，顯示日誌輸出，連接客戶端後將顯示一條成功的消息。

以相同的方式斷開與VPN的連接：右鍵單擊右下角的OpenVPN小圖示，選擇對應客戶端配置，然後點擊斷開連接。

安裝 Obfsproxy

現在，我們要構建Obfsproxy 。 Obfsproxy 需要使用Python （一種開源編程語言），因此您需要下載它。 轉到 www.python.org 並下載版本2.7 ，請使用默認選項進行安裝。

您還需要Visual C for Python，可以從 http://aka.ms/vcpython27 下載，下載後，請使用默認選項進行安裝。

現在，您已經安裝了Python ，現在就可以開始使用Obfsproxy。 打開 CMD shell窗口，然後輸入以下內容

  • cd C:\Python27\Scripts

現在，我們將通過輸入以下內容安裝Obfsproxy：

  • pip install obfsproxy

這將下載各種必需的模組並構建Obfsproxy這可能需要幾分鐘的時間…… 完成後，應該在 C:Python27\Scripts 中出現了可執行檔 obfsproxy.exe 。



配置 Obfsproxy

Obfsproxy不需要像Stunnel那樣的配置檔，因此我們現在只需配置OpenVPN即可。 下載 隱蔽的OpenVPN配置 檔，並將該檔複製到：

  • C:\Program Files\OpenVPN\config\


將OpenVPN客戶端與Obfsproxy連接

打開 CMD shell窗口，然後輸入以下內容：

  • cd C:\Python27\Scripts

輸入以下內容啟動Obfsproxy：

  • obfsproxy scramblesuit --dest 80.211.132.243:21343 --password=SSSTIZ3LG5SSS43OMSSSM4LKGQFASSSS client 127.0.0.1:21337


目標IP地址（* 在這種情況下為--dest 80.211.132.243）應該替換為您要連接的主節點的IP。我們僅在Alpha測試前階段使用此密碼。 未來會使用隨機字串代替密碼。


Obfsproxy啟動後，打開OpenVPN客戶端，右鍵單擊OpenVPN小圖示來啟動連接。 打開菜單，選擇菜單頂部的 veles-shielded （這是我們之前下載的 veles-shielded.ovpn 配置檔），然後 連接 。

建立連接後，將打開狀態窗口，顯示日誌輸出，連接客戶端後將顯示一條成功的消息。

以相同的方式斷開與VPN的連接：右鍵單擊右下角的OpenVPN小圖示，選擇對應客戶端配置，然後點擊斷開連接。

測試您的VPN連接

一旦所有的東西都安裝好了，只需要一個簡單的檢查就可以確認他是否運作。在未啟用VPN連接的情況下，打開流覽器並轉到 DNSLeakTest。
該站點將返回由您的網路服務提供商分配的IP地址，以及您在世界其他地方顯示的IP地址。若要通過同一網站檢查您的DNS設置，請單擊擴展測試（Extended Test），它會告訴您正在使用的DNS伺服器。
現在，連接到您的VPN客戶端並刷新流覽器，應該會顯示與您的VPN伺服器的完全不同的IP地址。然後再次進行擴展測試（Extended Test）檢查您的DNS設置，並確認您正在使用Veles dVPN推送的DNS解析。
