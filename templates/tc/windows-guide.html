Windows dVPN 教程

安裝

可在OpenVPN的下載頁上找到適用於您Windows的OpenVPN客戶端。

OpenVPN需要管理員許可權才能安裝。



Configuring OpenVPN

安裝OpenVPN後，下載配置檔
，並且將檔複製到你的配置檔夾中，例如：

  • C:\Program Files\OpenVPN\config

當您啟動OpenVPN時，它將自動尋找配置檔並啟動他。

OpenVPN每次使用時都必須以管理員身份運行，您不必每次使用OpenVPN時右鍵單擊並選擇以管理員身份運行，您可以對此進行預設，但這必須通過管理帳戶進行。這也意味著標準用戶需要輸入管理員密碼才能使用OpenVPN。另一方面，除非客戶端上的OpenVPN具有管理員許可權，否則標準用戶無法正確連接到伺服器，因此需要提升許可權。

如果要將OpenVPN設置為始終以管理員身份運行，請右鍵單擊其快捷方式圖示，然後選擇屬性。在相容性選項卡的底部，單擊按鈕為所有用戶更改設置。在新窗口中，選以管理員身份運行。



連接

每次啟動OpenVPN GUI時，Windows都會詢問是否允許程式對電腦進行更改單擊是（yes）。啟動OpenVPN客戶端應用程式後，他會隱藏在右下角託盤中，以便可以根據需要連接和斷開VPN，事實上他不會默認進行VPN連接。

進入右下角系統託盤並右鍵單擊OpenVPN圖示，啟動連接。這將打開菜單。在菜單頂部選擇veles（這是我們的veles.ovpn配置檔），然後選擇連接。

在建立連接時，將打開一個狀態窗口，顯示日誌輸出，並在連接客戶端後顯示一條消息。

以同樣的方式斷開與VPN的連接：進入右下角系統託盤，右鍵單擊OpenVPN小圖示，選擇客戶端配置檔，然後單擊斷開連接。

Installing Stunnel

Download and install Stunnel client by following this steps:

Stunnel needs administrative privileges to install.

Download official Stunnel client.

Click "Agree".

Choose “Install for anyone using this computer”.

Don’t changing anything here.

Choose directory for install Stunnel.

In the end of install, it will open new console window. You can input any information to these fields or let it blanc. It doesn’t matter and don’t have any affect to your connection.

Don’t mark “Start stunnel after installation", just click “Finish”.



Configuring Stunnel

Download Stunnel client config, Stunnel client certificate .

After you finish downloading , You will go into Stunnel directory , there you can remove all files and copy there veles.stunnel.conf and stunnel.pem

Probably you will need rename veles.stunnel.conf to stunnel.conf


  • C:\Program Files\Stunnel\config\

Now download Shielded OpenVPN config file and copy the file to:

  • C:\Program Files\OpenVPN\config\


Connecting OpenVPN client with Stunnel

At first turn on Stunnel. Once Stunnel is started turn on OpenVPN client and initiate a connection by going into the system tray applet and right-clicking on the OpenVPN applet icon. This opens the context menu. Select veles-shielded at the top of the menu (that’s our veles-shielded.ovpn profile) and choose Connect.

A status window will open showing the log output while the connection is established, and a message will show once the client is connected.

Disconnect from the VPN the same way: Go into the system tray applet, right-click the OpenVPN applet icon, select the client profile and click Disconnect.

Installing Obfsproxy

Now we are going build Obfsproxy. Obfsproxy requires Python, an open source programming language, so you’ll need to download it. Go to www.python.org and download version 2.7 and install it accepting all the defaults.

You’ll also need to get Visual C for Python, which you can download from http://aka.ms/vcpython27, once you have downloaded it, install it with the default options.

Ok now that you have Python installed, you’re ready to get obfsproxy. Open a CMD shell window and go to C:Python27Scripts by typing :

  • cd C:\Python27\Scripts

Now we will install Obfsproxy by typing:

  • pip install obfsproxy

This will download the various requisite modules and build obfsproxy – it might take a few minutes…… Once complete, you should have the obfsproxy.exe executable in C:Python27Scripts



Configuring Obfsproxy

Obfsproxy don't need some configuration files like in Stunnel so we will now just configure OpenVPN . Download Shielded OpenVPN config file and copy the file to:

  • C:\Program Files\OpenVPN\config\


Connecting OpenVPN client with Obfsproxy

Open a CMD shell window and go to C:Python27Scripts by typing:

  • cd C:\Python27\Scripts

Start Obfsproxy by typing:

  • obfsproxy scramblesuit --dest 80.211.132.243:21343 --password=SSSTIZ3LG5SSS43OMSSSM4LKGQFASSSS client 127.0.0.1:21337


The destination IP address (*in this case --dest 80.211.132.243) should be replaced with IP of masternode you are connecting. We will use this password only in pre-alpha testing phase. In future instead of password will be used random string.


Once Obfsproxy is started turn on OpenVPN client and initiate a connection by going into the system tray applet and right-clicking on the OpenVPN applet icon. This opens the context menu. Select veles-shielded at the top of the menu (that’s our veles-shielded.ovpn profile) and choose Connect.

A status window will open showing the log output while the connection is established, and a message will show once the client is connected.

Disconnect from the VPN the same way: Go into the system tray applet, right-click the OpenVPN applet icon, select the client profile and click Disconnect.

測試您的VPN連接

一旦所有的東西都安裝好了，只需要一個簡單的檢查就可以確認他是否運作。在未啟用VPN連接的情況下，打開流覽器並轉到 DNSLeakTest。
該站點將返回由您的網路服務提供商分配的IP地址，以及您在世界其他地方顯示的IP地址。若要通過同一網站檢查您的DNS設置，請單擊擴展測試（Extended Test），它會告訴您正在使用的DNS伺服器。
現在，連接到您的VPN客戶端並刷新流覽器，應該會顯示與您的VPN伺服器的完全不同的IP地址。然後再次進行擴展測試（Extended Test）檢查您的DNS設置，並確認您正在使用Veles dVPN推送的DNS解析。
