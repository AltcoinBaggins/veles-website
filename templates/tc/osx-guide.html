OS X dVPN 教程

安裝OpenVPN

Tunnelblick是一個免費的開源OpenVPN客戶端，適用於Mac OS X。您可以從Tunnelblick下載頁面下載最新的磁片映像。雙擊下載的.dmg檔並按照提示進行安裝。



配置OpenVPN

安裝過程快結束時，Tunnelblick將詢問您是否有任何配置檔，選擇否。下載Veles配置檔並雙擊Veles.ovpn，Tunnelblick將安裝客戶端配置檔（需要管理權限）。



連接OpenVPN

雙擊啟動Tunnelblick。啟動Tunnelblick後，螢幕右上角的菜單欄中將顯示一個Tunnelblick圖示，用於控制連接。單擊圖示，然後單擊連接菜單項啟動VPN連接。選擇veles進行連接。

Installing Stunnel

Press Command+Space and type Terminal and press enter/return key. Run in Terminal app and press enter/return key.

  • ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)" < /dev/null 2> /dev/null

If the screen prompts you to enter a password, please enter your Mac's user password to continue. When you type the password, it won't be displayed on screen, but the system would accept it. So just type your password and press ENTER/RETURN key. Then wait for the command to finish. After that type:

  • brew install stunnel


Configuring Stunnel

Download Stunnel client config, Stunnel client certificate .

After you finish downloading , You will go into Stunnel directory /usr/local/etc/stunnel/ and copy there downloaded veles.stunnel.conf and stunnel.pem

Probably you will need rename veles.stunnel.conf to stunnel.conf


  • ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)" < /dev/null 2> /dev/null

Download Veles configuration file and double-click veles-shielded.ovpn. Tunnelblick will install the client profile. Administrative privileges are required.



Connecting OpenVPN client with Stunnel

Press Command+Space and type Terminal and press enter/return key. Now we will turn on Stunnel by typing :

  • stunnel

Now launch Tunnelblick by double-clicking Tunnelblick in the Applications folder. Once Tunnelblick has been launched, there will be a Tunnelblick icon in the menu bar at the top right of the screen for controlling connections. Click on the icon, and then the Connect menu item to initiate the VPN connection. Select the veles-shielded connection.

測試您的VPN連接

一旦所有的東西都安裝好了，只需要一個簡單的檢查就可以確認他是否運作。在未啟用VPN連接的情況下，打開流覽器並轉到 DNSLeakTest。
該站點將返回由您的網路服務提供商分配的IP地址，以及您在世界其他地方顯示的IP地址。若要通過同一網站檢查您的DNS設置，請單擊擴展測試（Extended Test），它會告訴您正在使用的DNS伺服器。
現在，連接到您的VPN客戶端並刷新流覽器，應該會顯示與您的VPN伺服器的完全不同的IP地址。然後再次進行擴展測試（Extended Test）檢查您的DNS設置，並確認您正在使用Veles dVPN推送的DNS解析。
