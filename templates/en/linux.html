Linux dVPN Guide

Installing

If you are using Linux, there are a variety of tools that you can use depending on your distribution. Your desktop environment or network manager might also include connection utilities. The most universal way of connecting, however, is to just use the OpenVPN software.

On Ubuntu or Debian, you can install it just as you did on the server by typing:

  • sudo apt-get update
  • sudo apt-get install openvpn

On CentOS you can enable the EPEL repositories and then install it by typing:

  • sudo yum install epel-release
  • sudo yum install openvpn

Configuring

Download Veles configuration file and move to it's repository by typing:

  • wget https://explorer.veles.network/dapi/getOpenVPNConfig
  • sudo mv /etc/openvpn/client/veles.ovpn


To stay protected against DNS leaks use this command ( IP used in this case serve only as example and need to replaced by one from new Masternode list )

  • echo -e "#Veles decentralized DNS\nnameserver 111.111.111.111"

For GUI Users follow this steps.
At first click with right mouse button on network manager icon and and click on "Edit Connections..." like on image below:

Now we will press "+" symbol to add new connection

We will choose "Import a saved configuration file..." option for importing oru config

Now go to config location , in our case it will /etc/openvpn/client/ and pick veles.ovpn

You dont need to touch nothing it will automatically create Veles VPN profile for you just press "Save"

Connecting

For CLI Users follow this steps.
Now, you can connect to the VPN by just pointing the openvpn command to the client configuration file:

  • sudo openvpn --config /etc/openvpn/client/veles.ovpn

For GUI Users follow this steps.
Open Network Manager by clicking with right mouse button on it's icon. There you will see "VPN Connections" section. Click on it. Mark "veles" and enjoy your connection !

Test Your VPN Connection

Once everything is installed, a simple check confirms everything is working properly. Without having a VPN connection enabled, open a browser and go to DNSLeakTest.
The site will return the IP address assigned by your internet service provider and as you appear to the rest of the world. To check your DNS settings through the same website, click on Extended Test and it will tell you which DNS servers you are using.
Now connect to your VPN client and refresh the browser. The completely different IP address of your VPN server should now appear. That is now how you appear to the world.
