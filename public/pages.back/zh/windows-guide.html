Windows dVPN 教程

安装

可在OpenVPN的下载页上找到适用于您Windows的OpenVPN客户端。

OpenVPN需要管理员权限才能安装。



配置 OpenVPN

安装OpenVPN后，下载配置文件
，并且将文件复制到你的配置文件夹中，例如：

  • C:\Program Files\OpenVPN\config

当您启动OpenVPN时，它将自动寻找配置文件并激活他。

OpenVPN每次使用时都必须以管理员身份运行，您不必每次使用OpenVPN时右键单击并选择以管理员身份运行，您可以对此进行预设，但这必须通过管理帐户进行。这也意味着标准用户需要输入管理员密码才能使用OpenVPN。另一方面，除非客户端上的OpenVPN具有管理员权限，否则标准用户无法正确连接到服务器，因此需要提升权限。

如果要将OpenVPN设置为始终以管理员身份运行，请右键单击其快捷方式图标，然后选择属性。在兼容性选项卡的底部，单击按钮为所有用户更改设置。在新窗口中，选以管理员身份运行。



连接

每次启动OpenVPN GUI时，Windows都会询问是否允许程序对计算机进行更改单击是（yes）。启动OpenVPN客户端应用程序后，他会隐藏在右下角托盘中，以便可以根据需要连接和断开VPN，事实上他不会默认进行VPN连接。

进入右下角系统托盘并右键单击OpenVPN图标，启动连接。这将打开菜单。在菜单顶部选择veles（这是我们的veles.ovpn配置文件），然后选择连接。

在建立连接时，将打开一个状态窗口，显示日志输出，并在连接客户端后显示一条消息。

以同样的方式断开与VPN的连接：进入右下角系统托盘，右键单击OpenVPN小图标，选择客户端配置文件，然后单击断开连接。

安装 Stunnel

按照以下步骤下载并安装Stunnel客户端：

Stunnel需要管理权限才能安装。

下载官方的Stunnel 客户端 右键点击小图标 以管理员身份运行.

选择 "同意".

选择 “为任何人使用此计算机的安装” 。

在这块，什么都不要更改。

选择安装的目录。

在安装结束时，它会打开新的控制台窗口。 您可以输入任何信息，也可以留空。 怎么做都没关系，对您的连接没有任何影响。

不要勾选“安装完成开始使用隧道”，只需单击“完成” 就可以。



配置 Stunnel

下载 Stunnel 客户端配置, Stunnel 客户端证书 .

下载完成后，进入Stunnel配置目录，您可以删除所有文件，并且复制之前下载的文件到这个目录里面 veles.stunnel.conf和 stunnel.pem

可能需要将veles.stunnel.conf重命名为 stunnel.conf


  • C:\Program Files\Stunnel\config\

现在下载 隐蔽的OpenVPN配置 文件并且复制文件到:

  • C:\Program Files\OpenVPN\config\


将OpenVPN客户端与Stunnel连接

首先打开Stunnel。 然后打开OpenVPN客户端，右键单击OpenVPN小图标来启动连接。 打开菜单，选择菜单顶部的 veles-shielded （这是我们之前下载的 veles-shielded.ovpn 配置文件），然后 连接 。

建立连接后，将打开状态窗口，显示日志输出，连接客户端后将显示一条成功的消息。

以相同的方式断开与VPN的连接：右键单击右下角的OpenVPN小图标，选择对应客户端配置，然后点击断开连接。

安装 Obfsproxy

现在，我们要构建Obfsproxy 。 Obfsproxy 需要使用Python （一种开源编程语言），因此您需要下载它。 转到 www.python.org 并下载版本2.7 ，请使用默认选项进行安装。

您还需要Visual C for Python，可以从 http://aka.ms/vcpython27 下载，下载后，请使用默认选项进行安装。

现在，您已经安装了Python ，现在就可以开始使用Obfsproxy。 打开 CMD shell窗口，然后输入以下内容

  • cd C:\Python27\Scripts

现在，我们将通过输入以下内容安装Obfsproxy：

  • pip install obfsproxy

这将下载各种必需的模块并构建Obfsproxy这可能需要几分钟的时间…… 完成后，应该在 C:Python27\Scripts 中出现了可执行文件 obfsproxy.exe 。



配置 Obfsproxy

Obfsproxy不需要像Stunnel那样的配置文件，因此我们现在只需配置OpenVPN即可。 下载 隐蔽的OpenVPN配置 文件，并将该文件复制到：

  • C:\Program Files\OpenVPN\config\


将OpenVPN客户端与Obfsproxy连接

打开 CMD shell窗口，然后输入以下内容：

  • cd C:\Python27\Scripts

输入以下内容启动Obfsproxy：

  • obfsproxy scramblesuit --dest 80.211.132.243:21343 --password=SSSTIZ3LG5SSS43OMSSSM4LKGQFASSSS client 127.0.0.1:21337


目标IP地址（* 在这种情况下为--dest 80.211.132.243）应该替换为您要连接的主节点的IP。我们仅在Alpha测试前阶段使用此密码。 未来会使用随机字符串代替密码。


Obfsproxy启动后，打开OpenVPN客户端，右键单击OpenVPN小图标来启动连接。 打开菜单，选择菜单顶部的 veles-shielded （这是我们之前下载的 veles-shielded.ovpn 配置文件），然后 连接 。

建立连接后，将打开状态窗口，显示日志输出，连接客户端后将显示一条成功的消息。

以相同的方式断开与VPN的连接：右键单击右下角的OpenVPN小图标，选择对应客户端配置，然后点击断开连接。

测试您的VPN连接

一旦所有的东西都安装好了，只需要一个简单的检查就可以确认他是否运作。在未启用VPN连接的情况下，打开浏览器并转到 DNSLeakTest。
该站点将返回由您的网络服务提供商分配的IP地址，以及您在世界其他地方显示的IP地址。若要通过同一网站检查您的DNS设置，请单击扩展测试（Extended Test），它会告诉您正在使用的DNS服务器。
现在，连接到您的VPN客户端并刷新浏览器，应该会显示与您的VPN服务器的完全不同的IP地址。然后再次进行扩展测试（Extended Test）检查您的DNS设置，并确认您正在使用Veles dVPN推送的DNS解析。
